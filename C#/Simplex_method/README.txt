Данная программа представляет собой оконное приложение Windows написанное а языке C# с использование шаблонов WindowsForms.
Программа предназначенна для максимизации доходов компании. 
  Например, Предприятие может выпускать n видов продукции, используя для этого m видов ресурсов. Вам известна технологическая матрица A
затрат любого ресурса на единицу каждой продукции, вектор B объемов ресурсов и вектор C удельной прибыли.

Симплексный метод является методом упорядоченного перебора решений системы ограничений, при условии, что каждое следующее решение 
улучшает значение функции цели. 
Основной алгоритм находится в файле:  Simplex_prog/CalculationForm.cs
